# ILBench

Assembly benchmarking tool library.

## Features
- Works with console apps
- Works with ASP.NET Core apps
- Extensible design

## Structure
- `src/lib` - Core library
- `src/demo` - Console demo application

## Building
```
dotnet build src/lib
dotnet build src/demo
```

## Running Demo
```
dotnet run --project src/demo
```
